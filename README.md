## Overview

Customer behavior prediction and personalized marketing system using Large Language Models (LLMs), Generative AI, LangChain, and Llama2. Built at **Molina HealthCare** to enhance customer engagement and operational insights.

## Key Features

- **Customer Behavior Prediction** using Python and TensorFlow with LLMs for advanced feature extraction
- **ETL Pipelines** with Apache Airflow for processing large transaction and customer data volumes
- **Generative AI** for synthetic dataset creation addressing data privacy (CCPA/GDPR compliant)
- **Abstractive Summarization** of customer feedback using Llama2 and LangChain
- **Hyperparameter Tuning**: Grid search, random search, Bayesian optimization (15% accuracy improvement)
- **CI/CD Pipelines** with Jenkins for deployment to AWS EKS
- **Model Monitoring** with Prometheus and Grafana for bias detection
- **Streamlit Dashboard** for marketing and product team insights
